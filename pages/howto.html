Html

A tag html é utilizada para dizer onde um elemento começa, na gigantesca maioria dos casos, vem logo após a especificação do documento HTML.

body

Também conhecida como "corpo do texto" tudo o que vemos no site, imagens, links, ficam dentro dessa tag, ela é usada para nos comunicarmos com o usuário.

p

Essa tag é usada para representar um parágrafo, mas no Html os parágrafos tem outra utildade, eles são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de formulário.

pre

É utilizada para representar texto pré-formado um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo.

h1-h6

Os elementos HTML representam seis níveis de título de seção. É o nível de seção mais alto e é o mais baixo.

span

Ele pode ser usado para agrupar elementos para fins de estilo usando os atributos class ou id, ou para compartilhar valores de atributos como lang .

div

Ele pode ser utilizado para agrupar elementos para fins de estilos ou porque eles compartilham valores de atributos, como lang. Ele deve ser utilizado somente quando não tiver outro elemento de semântica.

Seletores CSS

Seletor de tags

É o seletor mais abrangente da CSS, ele casa com todos os elementos encontrados. Os estilos abaixo irão refletir em todos o elementos p e h1 da página.

Seletor por classes

Consiste na seleção de elementos por meio do nome da classe que foi definida como atributo e que tem como sintaxe o caractere.

Seletores de ID

Em um documento HTML, os seletores de ID do CSS selecionam um elemento baseado no conteúdo de seu atributo id, o qual deve ser exatamente igual ao valor dado ao seletor.
